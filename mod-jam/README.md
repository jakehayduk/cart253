# Mod Jam - The Frog Assassin

Jake Hayduk

[View this project online](https://jakehayduk.github.io/cart253/mod-jam/)

## Description

You are a very hungry, killer frog assassin that must kill the flies in front of him as fast as he possibly can!

> Use your mouse cursor to whack them with your long tongue!

> Aim for the lowest timer score possible.
